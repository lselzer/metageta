DMS2DD(dms, format)

source code 

Convert coordinate in degrees, minutes, seconds to decimal degrees.

Parameters:
  • dms (str) - degrees, minutes, seconds
  • format (str) - The format parameter is a string of equal length to the DMS string and identifies the position of each of the following elements. Any other string in the format parameter is ignored.
    • H - hemisphere
    • D - degrees
    • M - minutes
    • S - seconds

    eg:

    • '27 45 12 E' = 'DD MM SS H'
    • '027 45 12 E' = 'DDD MM SS H'
    • '027-45-12-E' = 'DDD MM SS H'
    • '27,45,12.3 E' = 'DD MM SSSS H'
    • '-27,45,12.3' = 'DDD MM SSSS'
Returns: float
decimal degrees

CreateMosaicedVRT(files, bands, srcrects, dstrects, cols, rows, datatype, relativeToVRT=0)

source code 

Create a VRT XML string that mosaics datasets.

For further info on VRT's, see the GDAL VRT Tutorial

Parameters:
  • files ([str,...,str]) - List of files to mosaic
  • bands ([int,...,int]) - List of band numbers (1 based). Eg. [1,2,3] will mosaic the first band from each file into the 1st band of the output VRT, etc.
  • srcrects ([SrcRect,...,SrcRect]) - List of SrcRects, one per file, in image not map units. E.g [[0,0,512,512],...] will be output as <SrcRect xOff="0" yOff="0" xSize="512" ySize="512"/>. The SrcRect allows you to subset your input image.
  • dstrects ([DstRect,...,DstRect]) - List of DstRects, One per file, in image not map units. E.g [[512,512,1024,1024],...] will be output as <DstRect xOff="512" yOff="512" xSize="1024" ySize="1024"/> The DstRect determines the spatial position of the input image in the mosaic.
  • cols (int) - The number of columns in the output mosaic
  • rows (int) - The number of rows in the output mosaic
  • datatype (str) - GDAL datatype name. Eg. Byte, Int32, UInt16
Returns: xml
VRT XML string
