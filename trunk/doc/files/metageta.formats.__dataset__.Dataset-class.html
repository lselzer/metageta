getoverview(self, outfile=None, width=800, format='JPG')

source code 

Generate overviews for generic imagery

Parameters:
  • outfile (string) - a filepath to the output overview image. If supplied, format is determined from the file extension
  • width (integer) - image width
  • format (string) - format to generate overview image, one of ['JPG','PNG','GIF','BMP','TIF']. Not required if outfile is supplied.
Returns:
  • filepath (if outfile is supplied) OR
  • binary image data (if outfile is not supplied)

Requires: exposing a gdal.Dataset object as self._gdaldataset

Note: There are a number of ways of controlling the generation of overview images:

  • Overriding this class method and writing your own.
  • Setting self._stretch to the appropriate (stretch_type,rgb_bands,stretch_args) values. See overviews.getoverview.
  • Customising self._gdaldataset using a VRT, for example: setting red, green and blue color interpretations for selected bands
