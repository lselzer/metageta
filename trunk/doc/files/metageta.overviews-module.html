getoverview(ds, outfile, width, format, bands, stretch_type, *stretch_args)

source code 

Generate overviews for imagery

Parameters:
  • ds (GDALDataset) - a GDALDataset object
  • outfile (str) - a filepath to the output overview image. If supplied, format is determined from the file extension
  • width (int) - output image width
  • format (str) - format to generate overview image, one of ['JPG','PNG','GIF','BMP','TIF']. Not required if outfile is supplied.
  • bands (list) - list of band numbers (base 1) in processing order - e.g [3,2,1]
  • stretch_type (str) - stretch to apply to overview image, one of [NONE,PERCENT,MINMAX,STDDEV,COLOURTABLE,COLOURTABLELUT,RANDOM,UNIQUE].
  • stretch_args (list) - args to pass to the stretch algorithms
Returns: str
filepath (if outfile is supplied)/binary image data (if outfile is not supplied)

ParseColourLUT(filepath)

source code 

Open and parse a colour lookup table

Values must be space separated. Comment characters are stripped out. Format is: pixel red green blue [optional alpha]. Pixel value may be a range (eg 15-20). The following comment characters may be used: '#' '//' ';' '/*'.

Example:

       #value   R   G   B   A
        11-15   255 0   0   255 #(red)
        16      255 165 0   255 #(orange)
        18      255 255 0   255 #(yellow)
        19      0   255 0   255 /*(green)
        21      0   0   255 255 /*(blue)
        98      0   255 255 255 ;(cyan)
        99-255  160 32  240 255 ;(purple)
Parameters:
  • filepath (str) - Path to colour lookup file
Returns: list
List of cell values and R,G,B,A values e.g ((12,0,0,0,255), (25,255,255,255,0))
